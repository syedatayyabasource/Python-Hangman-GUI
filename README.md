# 🎮 Hangman Pro - Python GUI Game

An interactive, professional Hangman game built with **Python** and **Tkinter**. This project demonstrates GUI development, dynamic vector graphics, and event-driven programming.

## 🚀 Key Features
- **Dynamic Graphics:** Real-time Hangman drawing using Tkinter Canvas.
- **Modern UI:** Clean, dark-themed aesthetic with a polished color palette.
- **Interactive Experience:** Supports keyboard 'Enter' key and auto-clearing inputs for smooth gameplay.
- **Smart Logic:** Includes input validation and duplicate guess prevention.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Library:** Tkinter (GUI)
- **Concepts:** Object-Oriented Programming (OOP), Vector Graphics, Event Handling.

## 📂 Installation & Usage

   ```bash
   git clone [https://github.com/syedatayyabasource/Python-Hangman-GUI.git](https://github.com/syedatayyabasource/Python-Hangman-GUI.git)
2.  Navigate & Run:
    cd Python-Hangman-GUI
    python hangman_gui.py
🎮 How to Play
The system selects a random secret word.

Enter one letter at a time in the input box and press Enter.

Each wrong guess draws a part of the hangman.

Guess the full word before the hangman is complete to win!
